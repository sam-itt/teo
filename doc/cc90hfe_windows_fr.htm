Usage de CC90HFE sous Windows

Normalement, le port série est détecté automatiquement par CC90HFE. Mais si tel n'est pas le cas, vous pouvez toujours forcer le nom du port série dans le fichier cc90hfe.ini qui sera créé après avoir lancé puis quitté CC90HFE, dans le répertoire teo en face de la clef 'port_name' dans la section '[serial]'.

Utilisation en ligne de commande

Utilisez le programme cc90hfe-com.exe ou bin/cc90hfe.exe que vous trouverez dans le répertoire racine de l'application. Utilisez l'option '-h' ou '--help' pour afficher le mode d'emploi.

Désinstallation

Si l'installation a été faite avec les fichiers zip, il suffit d'éliminer le répertoire cc90hfe racine.

Si l'installation a été faite avec le setup, demandez la désinstallation automatique dans le menu du programme.
