Le cordon DB25/DB9

Ce cordon sert à relier l'adaptateur USB/RS232 à l'interface CC90-232. Des cordons (ou des adaptateurs) existent dans le commerce. Il faudra vous procurer alors un adaptateur DB25 mâle/DB9 femelle entièrement câblé (pas de 'null modem'!).

sSi vous voulez vous lancer, voici la table des connexions (avec le numéro des broches):

204

Il est conseillé d'avoir un adaptateur le plus court possible. Si le câble n'est pas assez long, utilisez plutôt une rallonge USB côté USB-Série.
