Installation

Pour faire fonctionner l'émulateur, vous devez disposer au minimum d'un 80486 muni de 8 Mo de RAM, tournant sous Windows 98, Windows Millenium edition, Windows NT4, Windows 2000, Windows XP ou supérieurs; les composants redistribuables DirectX version 3.0 ou supérieure doivent être installés. Une carte son compatible DirectSound est nécessaire pour l'émulation sonore.

Vous pouvez installer l'émulateur de deux façons:

  • Exécutez teow-1.8.2-setup.exe.
  • Désarchivez teo-1.8.2-winexe.zip. Le programme éxécutable se nomme teow.exe.
Lancement

Teo accepte les paramètres suivants supplémentaires sur la ligne de commande:

--mode40lance l'émulateur en mode plein écran 40 colonnes 16 couleurs
--mode80lance l'émulateur en mode plein écran 80 colonnes 16 couleurs
--truecolorlance l'émulateur en mode plein écran 80 colonnes 4096 couleurs
--windowlance l'émulateur en mode fenêtré 80 colonnes 4096 couleurs (par défaut)
Joysticks

Teo utilise les joysticks PC disponibles pour émuler les 2 joysticks du TO8. Dans le cas où le nombre de joysticks détectés est inférieur à 2, le(s) joystick(s) manquant(s) est (sont) émulé(s) par le clavier.

Générateur sonore

Teo utilise le périphérique sonore par défaut du système (via DirectSound).
