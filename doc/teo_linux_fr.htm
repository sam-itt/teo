Installation

La configuration nécessaire pour faire tourner l'émulateur est la configuration standard des distributions récentes; seule la présence obligatoire de la version 3.0 ou supérieure du toolkit GTK+ peut nécessiter une mise à jour du système, disponible à http://www.gtk.org.

Vous pouvez installer l'émulateur de deux façons:

  • Exécuter teo-1.8.4-i586.deb. Le lancement de Teo se fera par le menu Application/Jeux/Emulateur Teo.
  • Extraire teo-1.8.4-i586.tar.gz. Le programme éxécutable se nomme teo.
Lancement

Le lancement de Teo s'effectue en cliquant sur l'icône de Teo. Si vous cliquez-droit dessus, vous pourrez choisir de simplement le lancer, ou de le lancer avec un reset à froid.

Clavier

Si la touche ALTGR reste sans effet, vérifez vos réglages pour l'agencement du clavier en ce qui concerne les touches de 3ème niveau: Agencement du clavier/Options.../Touche sélectionnant le 3ème niveau à partir des Paramètres Système.

Sur votre clavier est autre que AZERTY, vous pouvez toujours avoir accès au clavier Français en suivant la procédure suivante

  • Quitter Teo
  • Lancer setxkbmap fr dans un Terminal
  • Relancer Teo

Le seul problème est que pendant le temps où le clavier Français sera activé, il le sera pour toutes les applications. Pour revenir à votre clavier naturel, il suffira de lancer setxkbmap <votre_langage_clavier> dans un Terminal... ou de relancer votre distribution Linux.

Générateur sonore

Teo utilise le périphérique sonore par défaut du système (via ALSA).

Activer l'accès direct aux disquettes sur les systèmes Linux récents

Sur les systèmes récents, les lecteurs de disquette ne sont pas actifs à l'installation. Il est nécessaire de procéder à quelques ajustements à partir d'un Terminal :

Créer l'entrée pour le premier lecteur (si vous en avez un) :

  • Exécuter sudo mkdir /media/floppy0
  • Exécuter sudo gedit /etc/fstab
  • Ajouter la ligne /dev/fd0 /media/floppy0 auto rw,user,noauto,exec 0 0 si aucune ligne ne commence par /dev/fd0
  • Enregistrer le fichier et quitter GEdit

Créer l'entrée pour le deuxième lecteur (si vous en avez un) :

  • Exécuter sudo mkdir /media/floppy1
  • Exécuter sudo gedit /etc/fstab
  • Ajouter la ligne /dev/fd1 /media/floppy1 auto rw,user,noauto,exec 0 0 si aucune ligne ne commence par /dev/fd1
  • Enregistrer le fichier et quitter GEdit

Déclarer les accès disquette :

  • Exécuter sudo gedit /etc/modules
  • Ajouter la ligne floppy si elle n'existe pas
  • Enregistrer le fichier et quitter GEdit
  • Exécuter sudo adduser $USER floppy et éventuellement sudo adduser $USER disk (merci à credenhill de forum.ubuntu-fr.org)

Et pour finir, activer Le tout :

  • Redémarrer l'ordinateur

Si toutefois la lecture directe a des difficultés à fonctionner (erreurs d'entrée/sortie, par exemple), montez le lecteur de disquette en cliquant avec le bouton droit sur Lecteur de disquettes dans votre navigateur de fichiers et sélectionnez Détecter le média ou simplement en cliquant avec le bouton gauche (nécessite une disquette MSDOS dans le lecteur).

Désinstallation

Si l'installation a été faite avec les fichiers tar.gz, il suffit d'éliminer le répertoire teo racine.

Si l'installation a été faite avec les fichiers .deb, exécuter sudo apt-get remove --purge teo dans un Terminal.
