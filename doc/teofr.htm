Les options de lancement

Teo accepte ces principaux paramètres en ligne de commande:

[-m file.m7]place la cartouche file.m7 dans le lecteur
[-fast]active la vitesse rapide de l'émulateur
[-nosound]supprime le son de l'émulateur
[-loadstate]charge le dernier état sauvegardé de l'émulateur
Le panneau de contrôle

L'appui sur la touche [ESC] ouvre le Panneau de contrôle. Vous pouvez alors effectuer les opérations désirées.

Le clavier

Le caractère affiché sur l'écran est celui inscrit sur la touche PC frappée, pourvu qu'il existe sur le clavier de l'ordinateur Thomson. Les autres touches sont accessibles différemment :

STOPTAB
CNTCTRL gauche
ACCALT
DELBACKSPACE
F6F6 ou SHIFT+F1
F7F7 ou SHIFT+F2
F8F8 ou SHIFT+F3
F9F9 ou SHIFT+F4
F10F10 ou SHIFT+F5

Si vous avez des problèmes de réponse clavier, n'hésitez pas à jouer avec la touche de verrouillage du pavé numérique (Num Lock).

Les joysticks

En plus d'être éventuellement reconnus matériellement, il peuvent être contrôlés au clavier, Verr. Num. ou NumLock éteint :

Directions joystick 0Pavé numérique 7, 8, 9, 4, 5, 6, 1, 2, 3
Bouton joystick 0Ctrl droit
Directions joystick 1A, Z, E, Q, S, D, W, X, C
Bouton joystick 1Ctrl gauche
Le crayon optique ou la souris

Chacun de ces deux périphériques sera sélectionné dans le menu Réglage et Préférences du TO8. Ils sont tous les deux émulés par la souris PC.

Les disquettes

Les fichiers avec l'extension sap sont les disquettes virtuelles utilisées par Teo. Vous pouvez importer ou exporter vos disquettes avec le logiciel Sap2.

Vous pouvez aussi (selon le système) accéder directement à un floppy si vous avez un lecteur. Mais il est fortement recommandé de ne pas faire d'accès en écriture sur des disquettes auxquelles vous tenez. Formatez plutôt des disquettes 3"5 avec Teo lui-même et copiez-y vos originaux. Vous pourrez alors utiliser ces copies en toute sécurité.

Il est possible de se servir de disquettes 3"5 Haute Densité Double Face (1.44 Mo - 2HD) en occultant par un simple morceau de scotch l'encoche de haute capacité de la disquette.

Les cassettes

Les fichiers avec l'extension k7 sont les cassettes virtuelles utilisées par Teo. Vous pouvez importer ou exporter vos cassettes avec le logiciel Wav2k7.

Les cartouches

Les fichiers avec l'extension m7 sont les cartouches virtuelles utilisées par Teo.

Par défaut, chaque boîte de dialogue de fichier cherchera à s'ouvrir dans les répertoires disks pour les disquettes, k7 pour les cassettes et memo7 pour les cartouches.

Les imprimantes

Teo contient un module d'émulation des imprimantes matricielles Thomson PR90-055, PR90-600 et PR90-612 qui sait gérer à la fois la sortie brute, la sortie texte et la sortie graphique.

Commandes diverses

[F11] enregistre une capture d'écran (Windows)
[F12] affiche le débogueur (Linux et MS-DOS)
