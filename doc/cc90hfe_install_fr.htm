Installation de CC90HFE

Normalement, le port série est détecté automatiquement par CC90HFE. Mais si tel n'est pas le cas, vous pourrez toujours forcer le nom du port série dans le fichier cc90hfe.ini qui sera créé dès le premier lancement de CC90HFE, en face de la clef 'port_name' dans la section '[serial]'.
