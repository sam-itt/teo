1. Introduction

La version originale de l'émulateur Teo a été développée en 1994 sous MsDos avec la librairie Allegro. Pour s'adapter à l'évolution des OS, des portages ont été effectués pour win32 (Windows 95/98, XP, Vista et Seven) et pour Linux. Ces trois versions sont actuellement maintenues.

Par ailleurs un portage natif MacOS puis MacOSX est géré par R. Bannister depuis les toutes premières versions de TEO.

5. Différences entre versions

Les différences entre les trois versions sont réduites et concernent principalement quatre aspects du fonctionnement de l'émulateur: l'affichage graphique, le support des joysticks PC, l'accès direct aux disquettes Thomson et le chargement d'images de l'état de l'émulateur. Elles sont résumées dans le tableau ci-dessous: (X dénotant une caractéristique présente dans la version))

Affichage graphique Joysticks Accès direct Images
Plein écran Fenêtré   Lecteur 0: Lecteur 1:  
Version MSDOS/DPMI X   X X   X
Version Windows/DirectX X X X     X
Version Linux/X11   X   X X X

Les fonctions non présentes sur une plateforme peuvent être étudiées.

La lecture en accès direct ne fonctionnera pas avec un lecteur USB.
