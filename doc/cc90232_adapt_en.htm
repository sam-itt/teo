The DB25/DB9 cable

This cable is used to connect the USB-Serial adaptor to the CC90-232 interface. Those cables are available in stores. Don't take any "Null Modem" !

If you want to make it yourself, the connections with the number of pins below :

DB25DB9
23
32
47
58
75
204

We suggest you then to make the adaptor as short as possible. If it is not long enough, use an USB-Serial side extension instead.
