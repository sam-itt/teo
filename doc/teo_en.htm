1. Introduction

The original version of the Teo emulator was developed in 1994 for MSDos with the Allegro library. To adapt it to the OSes evolution, portings have been made for win32 (Windows 95/98, XP, Vista and Seven) and Linux. Those three versions are maintained by now.

Besides, a native MacOS and now MacOSX porting is managed by R. Bannister since the very first versions of Teo

5. Differences between versions

There are minor differences between the three versions. They affect the way the emulator behaves in four possible ways: graphic display, joystick support, direct access to Thomson floppy disks and support of emulation's state images. They are summed up in the following table (X represents a feature that exists in a version).

Graphic display Joysticks Direct access Images
Full screen Window mode   Floppy 0: Floppy 1:  
MSDOS/DPMI Version X   X X   X
Windows/DirectX Version X X X     X
Linux/X11 Version   X   X X X

The non present features on a platform may be considered.

The direct access won't work with an USB floppy drive.
