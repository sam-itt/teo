Usage de CC90HFE sous Linux

Normalement, le port série est détecté automatiquement par CC90HFE. Mais si tel n'est pas le cas, vous pouvez toujours forcer le nom du port série dans le fichier cc90hfe.ini - qui sera créé après avoir lancé puis quitté CC90HFE - dans le répertoire ~/.local/share/applications/teo ou le répertoire racine de l'application, en face de la clef port_name dans la section [serial].

Si l'accès au port série n'est pas autorisé, utiliser la commande Terminal sudo adduser $USER dialout et relancez Linux.

Utilisation en ligne de commande

cc90hfe peut être utilisé en ligne de commande. Utilisez l'option '-h' ou '--help' pour afficher le mode d'emploi.

Désinstallation

Si l'installation a été faite avec les fichiers tar.gz, il suffit d'éliminer le répertoire cc90hfe racine.

Si l'installation a été faite avec les fichiers .deb, exécuter sudo apt-get remove --purge cc90hfe dans un Terminal.
