# Pica Standard matrix for PR90-582 printer
#
# Matrix is as true as possible (cf PR90-582 documentation) but the position
#  of the character in the cell could be not exact
# ----------------------------------------------------------------------------
# Elite Standard is same but 10 points wide
# Double Width Standard is same but each point pair doubled and dot-width at 1
# ----------------------------------------------------------------------------
# Condensed Standard has special matrix
# ----------------------------------------------------------------------------
# Proportionnal Standard does not exist
# Italic Standard does not exist

# Font parameters :
# 0 : font width
# 1 : font height
# 2 : font fixed width
# 3 : font_elite_width
# 4 : dot width
# 5 : dot height
# 6 : double width 1 flag (1: creation of double width 1 needed)
# 7 : double width 2 flag (1: creation of double width 2 needed)
# 8 : condensed flag (1: creation of condensed needed)
# 9 : italic shift (1: creation of italic needed)
